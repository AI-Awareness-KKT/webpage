Game Control Using Hand Gestures

How Does the Game Control Work?

The Game Control project allows users to control games using hand gestures instead of traditional keyboard or mouse input. The system tracks the position of the index finger in real time and converts movements into directional controls.

By moving the finger up, down, left, or right, the system simulates arrow key presses, enabling hands-free game interaction.

Role of Computer Vision

Computer Vision processes the webcam feed and detects hand landmarks using real-time image analysis. The system identifies the index finger tip and tracks its position relative to the screen center.

Control zones are defined on the screen to determine directional movement, enabling accurate gesture-based input.

Role of Machine Learning

Machine Learning models help in detecting and tracking hand landmarks reliably. Pre-trained models such as MediaPipe Hands are used to recognize hand structure and finger positions.

This enables smooth and accurate gesture recognition even with small hand movements.

Key Features

  • Real-time hand gesture detection
  • Index finger-based directional control
  • Virtual arrow key simulation
  • Smooth gesture filtering for stability
  • Hands-free and interactive gameplay
